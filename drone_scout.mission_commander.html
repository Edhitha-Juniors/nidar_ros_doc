drone_scout.mission_commander module

This script defines the MissionCommander ROS 2 node.

It acts as a high-level state machine for controlling a MAVROS-enabled vehicle. The node listens for simple string commands (e.g., “mission_start”, “rtl”) from a user interface or another node, and translates them into the appropriate sequence of MAVROS service calls to achieve the desired action (e.g., changing mode, arming, taking off, and starting a mission).

class drone_scout.mission_commander.MissionCommander

Bases: Node

Commands a MAVROS-enabled vehicle based on high-level instructions.

This node subscribes to ‘/mission_commands’ and uses an asynchronous, callback-driven approach to interact with MAVROS services for vehicle control. The use of a MultiThreadedExecutor is crucial to handle the service calls and their responses without blocking the node.

Subscriptions:
/mission_commands (std_msgs.msg.String): Receives high-level commands like

“mission_start”, “rtl”, “land”, “disarm”.

MAVROS Service Clients:

/mavros/set_mode: To change the vehicle’s flight mode. /mavros/cmd/arming: To arm or disarm the vehicle. /mavros/cmd/takeoff: To command an autonomous takeoff. /mavros/mission/set_current: To start the mission from a specific waypoint.

arm(value=True)

Sends a request to MAVROS to arm or disarm the vehicle. :param value: True to arm, False to disarm.

disarm_vehicle()

Commands the vehicle to disarm.

handle_mission_start()

Initiates the mission start sequence: 1. Sets mode to GUIDED. 2. Arms the vehicle (upon successful GUIDED mode). 3. Commands takeoff (upon successful arming). 4. Sets mode to AUTO and starts mission from waypoint 0 (upon successful takeoff command).

land()

Commands the vehicle to land at its current position (LAND mode).

return_to_launch()

Commands the vehicle to return to its launch position (RTL mode).

set_current_waypoint(wp_seq)

Sets the current waypoint in an AUTO mission. :param wp_seq: The sequence number of the waypoint to set as current.

set_mode(mode_str)

Sends a request to MAVROS to change the vehicle’s flight mode. :param mode_str: The desired flight mode (e.g., “GUIDED”, “AUTO”, “RTL”, “LAND”).

takeoff(altitude=10.0)

Sends a takeoff command to MAVROS. :param altitude: The target altitude for takeoff in meters.

drone_scout.mission_commander.main(args=None)

Initializes and runs the MissionCommander node.

Uses a MultiThreadedExecutor to handle service client callbacks concurrently, preventing the node from blocking while waiting for service responses.
