drone_scout.backup_sender

This module implements the Radio Communication Backup Layer.

It enables:

  • Target coordinate transfer via MAVLink STATUSTEXT

  • Status broadcasting when internet is unavailable

  • Hybrid fallback for delivery missions

Used primarily by delivery drones operating in disconnected environments.

class drone_scout.backup_sender.TargetTransmitter

Bases: Node

Radio Backup Communication Node.

This node provides redundancy in target delivery coordination by:

  • Receiving targets through STATUSTEXT messages

  • Broadcasting delivery status back to the swarm

  • Supporting mission continuation without internet connectivity
