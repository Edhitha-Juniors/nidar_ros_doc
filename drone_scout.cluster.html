drone_scout.cluster module

drone_scout.cluster_scout

This module implements the Scout Drone Clustering + Drop Node.

The Scout Drone is responsible for:

  • Running surveillance missions

  • Performing object detection clustering

  • Maintaining a local registry of detected targets

  • Selecting confirmed targets for payload delivery

  • Executing autonomous drop operations

Modes Supported

  • Offline clustering from CSV logs

  • Live target registry mode via Ground Station Bridge

  • Manual override triggering

This node represents the central intelligence drone in the Edhitha swarm.

class drone_scout.cluster.Cluster(executor=None)

Bases: Node

Scout Drone Autonomous Drop + Target Selection Node.

This ROS 2 node acts as the central scout controller.

Responsibilities:

  • Transition from scanning to delivery mode

  • Fetch confirmed targets from the global registry

  • Select nearest target based on proximity sorting

  • Navigate to target location

  • Optionally perform vision-based centering

  • Descend and release payload using servo control

Unlike the delivery drone version, this node focuses on:

  • Global target coordination

  • Live clustering integration

  • Acting as the main swarm decision maker
