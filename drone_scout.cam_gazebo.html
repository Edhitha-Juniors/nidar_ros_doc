drone_scout.cam_gazebo module

This script defines a ROS 2 node that captures video frames from a GStreamer UDP stream and saves them as JPEG images. The image capture process is controlled by MAVROS waypoint updates, starting and stopping based on configured waypoint indices.

class drone_scout.cam_gazebo.GStreamerImageSaver

Bases: Node

A ROS 2 node to save images from a GStreamer pipeline based on MAVROS waypoints.

This node listens to a GStreamer UDP video stream. It subscribes to MAVROS waypoint messages to control when to start and stop saving frames. When active, it saves frames periodically and publishes metadata about each saved image.

Parameters:

  • folder_path (str) – The base directory where captured images will be saved.

  • start_waypoint_index (int) – The waypoint sequence number that triggers the start of image capture.

  • stop_waypoint_index (int) – The waypoint sequence number that triggers the end of image capture.

Subscriptions:

/mavros/mission/reached (mavros_msgs.msg.WaypointReached): Listens for reached waypoints to control capture state. /drone_status/last_waypoint_index (std_msgs.msg.String): Allows dynamic updating of the stop waypoint index.

Publications:

/camera/timestamps (std_msgs.msg.String): Publishes metadata for each saved image in the format “timestamp,filename”.

capture_frame()

Periodically captures a frame from the stream if capture is active.

This method is called by a timer. If the self.run flag is True, it reads a frame, saves it as a JPEG file with a nanosecond timestamp, and publishes metadata to the /camera/timestamps topic.

waypoint_callback(msg)

Controls the image capture state based on reached MAVROS waypoints.

Toggles the self.run flag to True when the start waypoint is reached and to False when the stop waypoint is reached.

Parameters:

msg (mavros_msgs.msg.WaypointReached) – The message from MAVROS indicating a waypoint has been reached.

waypoint_index_callback(msg: String)

Callback to dynamically update the stop waypoint index from a topic.

Parameters:

msg (std_msgs.msg.String) – The message containing the new stop waypoint index as a string.
